# Project Overview

This sample TM1 Planning Analytics project demonstrates a basic end-to-end workflow.

## Scope
- Create dimensions and cube
- Load source data using TI process
- Apply business rules
- Support reporting through Planning Analytics for Excel

## Components
- Cube: SalesCube
- Dimensions: Time, Region, Product, Version
- Data Source: CSV
- Logic: TI processes and Rules

## Business Use Case
This solution can be used for sales planning, budget tracking, and variance analysis.
