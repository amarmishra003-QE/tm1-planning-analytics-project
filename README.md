# TM1 Planning Analytics Project

This repository showcases a sample IBM Planning Analytics / TM1 implementation covering cube creation, data loading, rule logic, and technical documentation.

## Repository Structure

- `TI_Processes/` - Turbo Integrator process samples
- `Rules/` - TM1 rule files
- `Data/` - Sample input data files
- `Docs/` - Project and architecture documentation

## Functional Coverage

- Dimension creation
- Cube creation
- Data loading using TI
- Rule-based calculation
- Planning and reporting support

## Sample Components

- Cube: `SalesCube`
- Dimensions: `Time`, `Region`, `Product`, `Version`
- Input: CSV data
- Output: TM1 cube-based reporting

## Tools / Technologies

- IBM Planning Analytics (TM1)
- Turbo Integrator (TI)
- TM1 Rules
- Dimensions and Cubes
- SQL / ODBC
- PAfE

## Use Case

This project represents a simple sales planning and reporting use case where source data is loaded into TM1 and used for reporting and planning analysis.
